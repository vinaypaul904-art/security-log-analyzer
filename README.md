# Security Log Analyzer

This is a basic Python project that analyzes network logs and detects suspicious activities like repeated failed login attempts.

## Features
- Detects multiple failed login attempts
- Highlights suspicious IPs
- Simple and easy to understand

## Technologies
- Python

## How to Run

1. Run analyzer.py
2. Run dashboard.py
3. Open http://127.0.0.1:5000 in browser
